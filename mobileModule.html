Draw Something for iMessage

iMessage as a platform for games was brand new and documentation and related reference material were scarce. I learnt the Swift programming language for iOS development from scratch and iMessage, being a chat application, I had to consider various design and social elements to redesign the game, making for a simpler and seamless user experience. In the end I built a working prototype of the game with all core game functionality over a weekend.

Duel

Duel is a synchronous real-time PvP reaction based game that I developed for the Zynga India Hackathon. The game and the networking engine was written in native Java and the PvP element was achieved using socket communication. I replicated assets from the existing game, 'Ready Steady Bang' because I had to create the entire game over a weekend and I was working alone.

VIT Maps

I designed and created this App for my University. It consists of 3 modules. The Route Module was designed for people to find directions to a particular location within the University. The Location Finder Module was designed for visitors specifically. They could find all nearby Medical Stores, Restaurants, Book Stores, Hair Saloons, etc. The third Event Module was designed to help students receive real time notifications with information on upcoming events and where they were being held.
