Social MMO

This is a mass multiplayer platform through which players can create their own avatars and interact with each other in an SNES style fantasy world. It has been designed so that one player can host a session which can be joined by multiple clients. The platform was written in Java and can be looked at as a different take to Social Network. People can interact with each other and perform tasks in an open world while being completely anonymous.

Screen Shot of Two Instances of the Game. On the left is James' session, while the one on the right is Mary's session

The graphics module is responsible for reading and parsing the greyscale sprite sheet and then placing color over it, corresponding to the data it reads from the sprite sheet. This technique is efficient as designers will only need to create small greyscale levels when creating game maps.

The Multiplayer module provides a server and a client class. The server is set up and then, the clients can make requests to it. A UDP connection is established by exchange of IP Addresses and port numbers. A UDP connection was preferred over a TCP connection, because in the TCP connection, due to accuracy of packages being transferred, congestion occurs frequently which is undesirable. In UDP accuracy is compromised over efficient, fluid transfer of data.
