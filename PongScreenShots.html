Pong

To create this portable gaming console, I used an Arduino UNO onto which I soldered an Aadafruit 1.8" Color TFT Shield with a microSD and Joystick. You can find the Shield that I used here.

I wrote a minimalistic 'Breakout' game and ported it to the console. You can move the paddle across the screen using the joystick. Each time the ball makes contact with the paddle, you score a point and if you miss the game ends. The game comes with three difficulty levels ranging from easy to hard which players can toggle through using the joystick. From the screenshot you can see that the players current selection is highlighted in red. The paddle can be controlled using the joystick, and points are scored each time the player gets the ball to bounce off the paddle.

I'm someone who believes very strongly in the aesthetics of any product and so I designed a small User Interface and a little Pixel Artwork for the game.
